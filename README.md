## About

Messaging app for text, voice, and video communication.

## Features

- Text messaging
- Voice calls
- Video calls
- Group chats
- Media sharing

## Installation

### From App Store
Open the **App Store** in zOS and search for "WhatsApp".

### Via npm
```bash
npm install @zos-apps/whatsapp
```

## Usage

- Select chat to message
- Click call buttons
- Share media with +



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
